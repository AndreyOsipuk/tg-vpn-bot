# tg-vpn-bot

Telegram бот для продажи VPN (VLESS + Reality) через 3X-UI панель.

## Стек

TypeScript, Telegraf, SQLite, Docker, pino

## Оплата

ЮMoney (карта) — polling + ручная проверка

## Тарифы

| | Триал | Неделя | Месяц | Квартал | Год |
|---|---|---|---|---|---|
| RU | 0 | 50 | 150 | 400 | 1200 |
| NL | 0 | 80 | 250 | 650 | 2000 |

Трафик безлимитный.

## Архитектура

```
VDS #1 (RU) — бот + 3X-UI
VDS #2 (NL) — 3X-UI
```

Бот и RU VPN на одном сервере (через localhost). При росте нагрузки можно вынести бота на отдельный VDS.

## Запуск

```bash
cp .env.example .env
# заполнить .env
docker compose up -d --build
```

## Команды бота

- `/start` — выбор сервера и покупка
- `/status` — мои подписки
- `/keys` — ссылки для подключения
- `/support` — написать в поддержку
- `/help` — справка

## Админ

- `/admin` — панель
- `/stats` — статистика
- `/users` — активные юзеры
- `/health` — статус серверов
- `/block <id>` — заблокировать юзера
- `/broadcast <текст>` — рассылка
- `/toggle_sales` — вкл/выкл продажи
